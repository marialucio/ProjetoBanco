# PROJETO BANCO

Aplicação console para saques, depositos e tranferências bancárias.

## Funcionalidades

- Criar conta bancária.
- Sacar.
- Depositar.
- Transferir.

## Tecnologias utilizadas
 
- dotnet 6.0

## Como executar
 Realize o clone do repositório e execute o comando:
 
 ```
 dotnet run 
 ```
